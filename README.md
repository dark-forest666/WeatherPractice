# 🌤️ 高德天气 Android 应用

一个基于高德地图开放平台天气 API 的 Android 天气应用。

## 功能
- 输入城市中文名（如“北京”、“杭州”）
- 自动获取实时天气、温度、湿度、风力
- 使用 Retrofit + LiveData + MVVM 架构

## 使用说明
1. 在 [高德开放平台](https://console.amap.com/dev/key/app) 申请 Web 服务 Key
2. 替换 `MainActivity.java` 中的 `YOUR_API_KEY_HERE`
3. 运行 App

## 技术栈
- Java
- Android SDK
- Retrofit 2
- Gson
- Material Design
